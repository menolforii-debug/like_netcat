# cms.devel9.ru

CMS с Codex-архитектурой, вдохновлённой идеологией NetCat.

Корень сайта: `public_html`  
Для администрирования используется отдельная панель `public_html/admin.php`.

Основные документы:
- CODEX.md / CODEX.ru.md
- ARCHITECTURE.md / ARCHITECTURE.ru.md
- codex.yaml (нормативный)

## Модель ролей

В админ-панели используются две роли:

- `admin` — полный доступ ко всем разделам и настройкам.
- `editor` — доступ только к работе с контентом в инфоблоках и объектами (создание, редактирование, удаление, публикация).
