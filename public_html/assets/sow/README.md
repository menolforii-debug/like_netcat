# StepOfWeb Smarty (SOW) Assets

Скопируйте файлы дистрибутива SOW (Smarty UI kit) в папку `public_html/assets/sow/`.

Ожидаемая структура:

```
public_html/assets/sow/
  css/
    core.min.css
    vendor_bundle.min.css
  js/
    core.min.js
    vendor_bundle.min.js
```

Если файлы отсутствуют, интерфейс автоматически использует CDN Bootstrap 5.

> Важно: не добавляйте сюда файлы с лицензией в репозиторий.
