# StepOfWeb Smarty (SOW) Assets

Скопируйте файлы из приобретённого пакета StepOfWeb Smarty в папку `public_html/assets/sow/`.

Ожидаемая структура:

```
public_html/assets/sow/
  css/
    core.min.css
    vendor_bundle.min.css
  js/
    core.min.js
    vendor_bundle.min.js
  fonts/
    *
  images/
    *
```

Если файлы отсутствуют, интерфейс автоматически использует CDN Bootstrap 5.

> Важно: не добавляйте сюда лицензионные файлы SOW в репозиторий.
