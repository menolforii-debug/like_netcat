# Архитектура cms.devel9.ru

## Корень сайта

Единственная публичная директория — `public_html`.

## Жизненный цикл запроса

HTTP-запрос
→ index.php
→ определение раздела
→ (опционально) объект
→ SEO
→ рендеринг

## Рендеринг

Шаблоны:
templates/{component}/{view}.php

Шаблоны отвечают только за отображение.
